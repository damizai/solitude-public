投诉

内置的外挂标签满足大部分用户的需求，不需要额外安装插件。
插件的外挂标签需要安装对应的插件，满足部分用户的特殊需求。

内置标签

Article

文章列表标签，用于显示文章列表。

参数 说明 默认值
path 文章路径 无

Chart

图表标签，用于显示图表。
打开配置文件中开关

不需要参数。

Gallery

图片列表标签，用于显示图片列表。

GalleryGroup

图片组标签，用于显示图片组。

参数 说明 默认值
name 图片组名称 无
desc 图片组描述 无
url 图片组链接 无
img 图片组预览图 无

Mermaid

Mermaid 标签，用于显示 Mermaid 图表。

Typeit

Typeit 标签，用于显示 Typeit 动画。

参数 说明 默认值
div 标签名 无

Tabs

标签页标签，用于显示标签页。

基础

插件标签

安装插件

Youtube

Youtube 标签，用于显示 Youtube 视频。

参数 说明 默认值
video_id 视频 ID 无
type 视频类型 video
cookie 是否使用 cookie false

Bilibili

Bilibili 标签，用于显示 Bilibili 视频。

参数 说明 默认值
video_id 视频 ID 无

Audio

Audio 标签，用于显示音频。

参数 说明 默认值
url 音频链接 无

Video

Video 标签，用于显示视频。

参数 说明 默认值
url 视频链接 无

Videos

Videos 标签，用于显示视频列表。

参数 说明 默认值
col 列数 1

P

没错，就是 P 标签。

参数 说明 默认值
cls 类名 无
content 内容 无

cls: center, left, right, h1, h2, h3, h4, h5, h6, red, green, blue, yellow, purple, orange, gray

Hello World!

Span

没错，就是 Span 标签。

参数 说明 默认值
cls 类名 无
content 内容 无

Hello World!

Fold

折叠标签，用于显示折叠内容。

参数 说明 默认值
title 标题 无
open 是否展开 不填不显示
title

content

Img

图片标签，用于显示图片。

参数 说明 默认值
url 图片链接 无
alt 图片描述 无
style 图片样式 无

Inline Img

内联图片标签，用于显示内联图片。

参数 说明 默认值
url 图片链接 无
alt 图片描述 无
style 图片样式 无

Note

Note 标签，用于显示提示。

参数 说明 默认值
type 类型 无
icon 图标 无

type: primary, success, warning, danger, info, modern, flat, simple
icon: fantawesome 图标（例如： fas fa-mail）
